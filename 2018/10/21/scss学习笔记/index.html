scss学习笔记

发表于 2018-10-21 更新于 2022-11-07 分类于 scss Disqus：

&符号代表当前元素

#main {
    color: black;
    a {
        font-weight: bold;
        &:hover {
            color: red;
        }
    }
}
// is compiled to
#main {
    color: black;
}
#main a {
    font-weight: bold;
}
#main a:hover {
    color: red;
}
