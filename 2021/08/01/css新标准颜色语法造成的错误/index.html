css 新标准下的颜色语法

css 新标准下透明度也可写在 rgb 方法中，且可以不写逗号如rgb(0 0 0 / 100%)，该标准已被最新的浏览器实现，但尚未被 CSS 预处理器如 sass 和微信小程序实现，故暂时不推荐使用最新的写法，会造成预处理器编译错误和小程序白屏。
